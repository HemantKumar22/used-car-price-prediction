# 🚗 Used Car Price Prediction

## Project Overview
This project predicts the selling price of used cars using Machine Learning algorithms. The goal is to help buyers and sellers estimate fair market prices based on vehicle characteristics.

## Business Problem
Determining the correct price for a used car is challenging due to factors such as age, mileage, brand, and fuel type. This project uses data-driven techniques to improve pricing accuracy.
